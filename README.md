# desbloqueando-logica-python
Projeto em Python que segue um jeito simples de aprender: entender, criar, confirmar e mostrar. A ideia é transformar lógica em código de forma clara e prática, sem complicar. Feito para iniciantes e também para quem já programa, mas ainda quer organizar melhor o raciocínio e ganhar mais confiança no que está fazendo.
